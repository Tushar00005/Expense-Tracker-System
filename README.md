ExpenseTrackerSystem

A modern full-stack Expense Tracker application designed to help users manage budgets, monitor expenses, and analyze spending habits efficiently through 
interactive dashboards and analytics.

Features:
User Authentication & Secure Login
Monthly Budget Management
Expense & Transaction Tracking
Category-Based Spending Analysis
Interactive Charts & Analytics
Necessary vs Waste Expense Monitoring
Dark / Light Mode Support
Responsive Modern UI
Edit & Delete Transactions
Real-Time Financial Insights
Tech Stack
Frontend (Client)
React.js
Vite
Tailwind CSS
Recharts
Axios
Backend (Server)
Node.js
Express.js
MongoDB
JWT Authentication
Mongoose
Installation
Clone Repository
git clone https://github.com/your-username/ExpenseTracker.git

Client Setup:
cd client
npm install
npm run dev

Server Setup:
cd server
npm install
npm start
